# TAL1_noncoding_publication

This is a repo for the manuscript:

"Reactivation of a TAL1 progenitor cell enhancer region by non-coding somatic variants in T-lineage acute lymphoblastic leukemia".

* Fig1: Overview of TAL1 alterations in T-ALL and study design for somatic variants at downstream enhancer I region.

* Fig5: Predictions of TAL1 variants impact with AlphaGenome.

* Fig6: Molecular features of the TAL1 downstream enhancer I group.

* Fig7: Schematic of TAL1 enhancer reactivation vs neo-enhancer models.
